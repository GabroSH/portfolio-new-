მე გავდივარ ვებ-დეველოპმენტის კურსებს და ვიღებ გამოცდილებას სხვადასხვა პროექტებზე მუშაობით. 1 წელია ალბათ რაც დაინტერესებული ვარ, მაგრამ სწავლა უშუალოდ რამდენიმე თვეა რაც დავიწყე. ჩემი მიზანია დავეუფლო HTML, CSS და JS.

სწავლების საგნები

HTML

ჰიპერტექსტის მარკირების ენა ან HTML არის სტანდარტული მარკირების ენა დოკუმენტებისთვის, რომლებიც შექმნილია ვებ ბრაუზერში გამოსატანად. მას შეიძლება დაეხმარონ ისეთი ტექნოლოგიები, როგორიცაა Cascading Style Sheets (CSS) და სკრიპტირების ენები, როგორიცაა JavaScript. HTML.

CSS

CSS არის ენა, რომელსაც ვიყენებთ HTML დოკუმენტის სტილისთვის. CSS აღწერს, თუ როგორ უნდა იყოს ნაჩვენები HTML ელემენტები.

JS

Javascript (JS) არის სკრიპტირების ენა, რომელიც ძირითადად გამოიყენება ინტერნეტში. იგი გამოიყენება HTML გვერდების გასაუმჯობესებლად და ჩვეულებრივ გვხვდება HTML კოდში ჩასმული. JavaScript არის ინტერპრეტირებული ენა. ამრიგად, მისი შედგენა არ არის საჭირო. JavaScript ასახავს ვებ გვერდებს ინტერაქტიული და დინამიური ფორმით.
